Create a simple mono or stereo file editor, think a stripped back version of audacity, that runs in the browser. 

Main function: to do simple edits on existing wave and aif files. No recording necessary

User can do the following: 
Use transport 
Import file 
Export file - set file format and quality
Zoom in and out on audio waveform (including via shortcuts, shift scrolling or similar and buttons) 
Trim - this removes audio outside of the current selection and moves the selected audio so it start at 0
Normalise - with option to set normalisation peak in dbfs 
Fade in / out based on selection
Gain - apply gain in dB

This should be a browser-based page
